С помощью Git программисты и разработчики ориентируются в коде и отслеживают изменения. Git помогаетвернуть файлы в исходное состояние и видеть изменения, внесённые в определённый период. Разработчик выполняет разные команды, а все изменения синхронизируются с центральным репозиторием. Git - это система контроля версий, а GitHub - онлайн-сервис, по сути социальна сеть. Одна из основных целей GitHub - единым местом для проектов с исходным кодом. Предполагается, что пользовател делится чем-то полезным, а другие люди смогут участвовать в разработке.
